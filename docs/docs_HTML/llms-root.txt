# PMD Core Ontology (PMDco)

> PMDco is a mid-level, BFO-aligned ontology for materials science and engineering (MSE), developed by the Platform MaterialDigital (PMD). It provides reusable classes, properties, usage patterns and SHACL shapes for representing materials, processes, measurements and simulations in knowledge graphs.
