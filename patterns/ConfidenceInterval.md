# Pattern 1: confidence interval composed of confidence level, lower confidence limit and upper confidence limit
## Purpose
Description on how to represent confidence interval of a measurement.

## Description
A confidence interval is a data item which defines an range of values in which a measurement or trial falls corresponding to a given probability.
To specify the confidence level and lower confidence limit and upper confidence limit, it is recommended to use the property:

```
has_part (BFO:0000051)
```
